# 📢 Campus Connect – Smart College Communication System

## 🚀 Project Overview
Campus Connect is a web-based application designed to improve communication between college administration and students. It provides a centralized platform where admins can post announcements and students can view them in real-time.

---

## ❗ Problem Statement
Students often miss important updates such as exams, events, and holidays due to scattered communication systems like notice boards, WhatsApp groups, or emails.

---

## 💡 Solution
Campus Connect solves this problem by creating a single platform for all announcements, ensuring students never miss important information.

---

## 🔥 Features
- 🔐 Role-based login (Admin & Student)
- 📢 Admin can add announcements
- 👨‍🎓 Students can view updates
- 🗄️ Data stored in database (SQLite)
- ⚡ Real-time updates

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS
- Backend: Python (Flask)
- Database: SQLite
